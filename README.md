# 📚 مكتبة طيبة (TAYBAA-LIBRARY)

موقع مكتبة إلكترونية بسيط وسهل الاستخدام: تصفح الكتب أونلاين أو حمّلها على جهازك.

## ✨ المزايا

- 🔍 **بحث فوري** في العنوان والمؤلف والقسم
- 📂 **تصنيف الكتب** حسب الأقسام بأيقونات واضحة
- 📖 **قراءة أونلاين** بدون تحميل
- ⬇️ **تنزيل مباشر** بصيغة PDF
- 👁️ **عدّاد المشاهدات والتحميلات** (Firebase اختياري)
- 📱 **تصميم متجاوب** يعمل على الجوال أولاً
- 🌐 **دعم RTL كامل** للعربية
- 🚀 **بدون خادم** — موقع ثابت 100%

## 🚀 البدء السريع

```bash
python3 -m http.server 8080
```
ثم افتح: <http://localhost:8080>

## ⚙️ الإعداد الكامل

راجع ملف [`SETUP.md`](./SETUP.md) لـ:
- ربط Google Sheets كقاعدة بيانات
- إعداد Firebase للعدّاد
- النشر مجاناً على Netlify / GitHub Pages

## 📁 هيكل الملفات
```
.
├── index.html              # الصفحة الرئيسية
├── book.html               # صفحة تفاصيل الكتاب

├── css/style.css           # التنسيق
├── js/
│   ├── config.js           # ⚙️ الإعدادات (عدّل هنا)
│   ├── data.js             # تحميل الكتب

│   ├── app.js              # منطق الرئيسية
│   └── book.js             # منطق صفحة الكتاب
├── data/books-sample.json  # بيانات تجريبية
├── README.md

```
