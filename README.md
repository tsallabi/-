# 📚 مكتبة ليبيا الطيبة — TAYBAA-LIBRARY

مكتبة إلكترونية حديثة وبسيطة، تتيح للقراء تصفح الكتب أونلاين أو تحميلها، مع لوحة إدارة احترافية لإضافة الكتب.

## ✨ الميزات الرئيسية

### للقراء:
- 🔍 بحث فوري في العنوان والمؤلف والقسم
- 📂 تصنيف بصري للأقسام بأيقونات
- 📖 قراءة أونلاين بدون تحميل
- ⬇️ تنزيل مباشر بصيغة PDF
- 🌗 وضع داكن قابل للتبديل
- 📱 متجاوب بالكامل (Mobile-first)
- 🎨 تصميم حديث مع Glassmorphism وحركات ناعمة

### للأدمن (لوحة الإدارة):
- 🔐 تسجيل دخول بكلمة مرور
- ➕ إضافة/تعديل/حذف الكتب
- 📊 إحصائيات المشاهدات والتحميلات
- 📝 محرّر كتب من Word: ارفع .docx يتحوّل تلقائياً لكتاب منسّق
- 🖼️ رفع الأغلفة بالسحب والإفلات
- 📄 رفع PDF أو استخراج من Word
- ✨ استخراج تلقائي للعنوان والنبذة والمقدمة من ملف Word

## 🚀 البدء السريع

### على الكمبيوتر:
```bash
python3 -m http.server 8080
```

### على Android (Termux):
```bash
pkg install python git -y
git clone https://github.com/tsallabi/TAYBAA-LIBRARY.git
cd TAYBAA-LIBRARY && python -m http.server 8080
```

افتح: <http://localhost:8080>

## 🔐 الدخول إلى لوحة الإدارة

افتح: **`your-site/admin.html`**

كلمة المرور الافتراضية: **`taybaa2026`** (غيّرها في `js/config.js`).

## ⚙️ الإعداد الكامل

راجع [`SETUP.md`](./SETUP.md) لـ:
- إعداد Firebase (للحفظ الدائم)
- تخصيص كلمة مرور الأدمن
- إضافة كتب من Word/Google Docs
- النشر على Netlify / GitHub Pages

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|----------|
| HTML/CSS/JS | الواجهة (بدون إطارات) |
| Firebase Firestore | تخزين الكتب |
| Firebase Storage | تخزين الأغلفة والـ PDFs |
| Mammoth.js | تحويل Word → HTML |
| Google Fonts (Cairo + Amiri) | الخطوط العربية |
